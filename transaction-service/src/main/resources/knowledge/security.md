# PayVora Account & Security Management

## Overview
PayVora employs bank-grade security protocols including Multi-Factor Authentication (MFA), biometric login, 6-digit Transaction PINs, and automated fraud protection monitoring.

## Key Security Features
- **Transaction PIN**: Required to authenticate outgoing transfers, withdrawals, and bill payments.
- **Login Security & Password Reset**: Secure password policy and instant email password reset links.
- **Account Lock & Fraud Protection**: Automatic temporary lock upon multiple failed PIN or login attempts to safeguard funds.
- **Lost Phone & Device Verification**: Deauthorize unknown devices instantly via Profile Security.

## Resetting Forgot Transaction PIN
If you forgot your Transaction PIN:
1. Navigate to **Security Settings** under your account profile.
2. Select **Reset Transaction PIN**.
3. Verify your identity via email/SMS OTP verification to set a new PIN.
